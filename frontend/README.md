# Frontend

Bu klasör React tabanlı kullanıcı arayüzünü içerir.

## Hedef

- React uygulaması
- Dashboard, Documents, Generate Tests, Results, History sayfaları
- API çağrıları için temel client
- Export butonları ve sonuç görüntüleme

## Önerilen klasör yapısı

- `public/`
- `src/pages/`
- `src/components/`
- `src/api/`
- `src/styles/`
- `package.json`
- `tsconfig.json`
