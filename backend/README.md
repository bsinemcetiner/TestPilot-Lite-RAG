# Backend

Bu klasör FastAPI tabanlı backend servislerini içerir.

## Hedef

- FastAPI uygulaması
- Sağlık kontrol endpoint'i
- Document upload endpointleri
- Chunking, embedding, vector store, retrieval ve test generation servislerinin altyapısı
- SQLite metadata ve history desteği

## Önerilen klasör yapısı

- `app/api/`
- `app/services/`
- `app/models/`
- `app/db/`
- `requirements.txt`
